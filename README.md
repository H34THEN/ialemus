# Ialemus

Android music and media player for HiBy R4 DAP, Android phones, and self-hosted NAS media workflows.

Ialemus is designed to connect with a Ugreen NAS media stack, including spot-dl, MeTube, Soulseek/slskd, local storage, NAS storage, play counts, favorites, lyrics, themes, widgets, and landscape-first playback layouts.
